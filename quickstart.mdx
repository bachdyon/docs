Trang này dành cho người mới, không rành terminal, không biết phải cài gì trước.

## Repo này là gì?

Repo này là một bộ dụng cụ để AI agent hỗ trợ bạn làm video ngắn.

Bạn đưa cho agent kịch bản, video thô, ảnh, audio hoặc yêu cầu sáng tạo. Agent sẽ đọc hướng dẫn trong repo này để biết nên phân tích asset, chọn cảnh, tạo giọng đọc, làm phụ đề, dựng video và render theo thứ tự nào.

Nói đơn giản: đây không phải là một app có nút bấm sẵn. Đây là một “workspace” để bạn mở bằng Cursor, Claude, Codex, Antigravity hoặc app AI tương tự, rồi giao việc bằng prompt.

<Info>
  Nếu bạn chưa biết nên bắt đầu từ đâu, chỉ cần tải repo, mở đúng thư mục bằng app AI, rồi copy prompt đầu tiên trong trang này.
</Info>

Bạn chỉ cần làm 3 việc:

1. Tải repo này về máy.
2. Mở đúng thư mục repo bằng Cursor, Claude, Codex, Antigravity hoặc app AI bạn đang dùng.
3. Copy prompt mẫu bên dưới và gửi cho AI agent.

<Info>
  Đừng tự cài lung tung từ đầu. Hãy để AI agent đọc repo, kiểm tra máy, rồi nói rõ bạn cần cài gì cho đúng tính năng bạn muốn dùng.
</Info>

## 1. Tải repo về máy

Cách dễ nhất:

1. Mở trang GitHub của repo: [video-automator-skills](https://github.com/bachdyon/video-automator-skills).
2. Bấm nút **Code** màu xanh.
3. Chọn **Download ZIP**.
4. Giải nén file vừa tải.
5. Bạn sẽ có một thư mục tên gần giống `video-automator-skills`.

<Tip>
  Nếu tên thư mục có thêm chữ `-main` ở cuối cũng không sao. Quan trọng là bên trong thư mục đó có file `AGENTS.md`.
</Tip>

<AccordionGroup>
  <Accordion title="Nếu bạn đã biết dùng Git">
    Bạn có thể clone repo thay vì tải ZIP:

    ```bash
    git clone https://github.com/bachdyon/video-automator-skills
    ```

    Cách này tiện hơn khi muốn cập nhật repo sau này.
  </Accordion>
</AccordionGroup>

## 2. Mở thư mục bằng AI agent

Trong app AI bạn dùng, hãy chọn **Open Folder**, **Open Project**, hoặc **Open Workspace**, rồi chọn thư mục `video-automator-skills`.

<Warning>
  Đừng chọn riêng thư mục `docs/`, `skills/`, `setup/`, hoặc `raw_assets/`. Hãy chọn thư mục lớn bên ngoài, nơi có file `AGENTS.md`.
</Warning>

## 3. Copy prompt đầu tiên

Sau khi mở repo bằng AI agent, copy prompt này và gửi cho agent:

```text
Tôi là người mới, không rành terminal và không biết repo này cần cài gì.

Hãy đọc AGENTS.md và docs/features.mdx trước.
Sau đó giải thích thật đơn giản:
1. Repo này làm được gì?
2. Nếu tôi muốn dùng từng tính năng thì cần chuẩn bị gì?
3. Máy tôi hiện đã sẵn sàng đến đâu?
4. Tôi nên bắt đầu bằng tính năng nào?

Đừng chạy lệnh cài đặt ngay.
Nếu cần cài gì, hãy giải thích bằng ngôn ngữ dễ hiểu và hỏi tôi xác nhận trước.
```

Nếu agent trả lời quá kỹ thuật, gửi tiếp prompt này:

```text
Hãy giải thích lại cho người không rành công nghệ.
Đừng dùng thuật ngữ nếu không cần.
Nếu bắt buộc dùng thuật ngữ, hãy giải thích bằng một câu ngắn.
```

## 4. Hỏi repo có thể làm gì

Bạn có thể copy từng prompt dưới đây, tuỳ việc bạn muốn làm.

### Xem toàn bộ tính năng

```text
Hãy liệt kê các tính năng của repo này theo nhóm dễ hiểu:
- tải video/ảnh/audio,
- tìm kiếm asset,
- tạo kịch bản,
- tạo giọng đọc,
- làm phụ đề,
- dựng video,
- render video,
- upload/chia sẻ file.

Với mỗi nhóm, hãy nói tôi cần chuẩn bị gì và có cần API key không.
```

### Kiểm tra máy cần cài gì

```text
Hãy kiểm tra máy của tôi đã dùng được tính năng nào trong repo này.

Chia kết quả thành 3 nhóm:
1. Dùng được ngay.
2. Cần cài thêm nhưng dễ.
3. Cần API key hoặc setup nâng cao.

Đừng cài gì khi chưa hỏi tôi.
```

### Dùng Asset Index để tìm video/ảnh

```text
Tôi muốn tìm kiếm ảnh/video trong raw_assets bằng tiếng Việt.
Hãy hướng dẫn tôi chuẩn bị tối thiểu để dùng Asset Index.
Nếu cần chạy installer, hãy nói rõ installer làm gì rồi hỏi tôi xác nhận.
```

### Làm video từ file có sẵn

```text
Tôi đã có video/ảnh/audio muốn dùng trong raw_assets.
Hãy hướng dẫn tôi làm một video short-form từ các file đó.
Trước tiên hãy hỏi tôi vài câu đơn giản về chủ đề, phong cách, độ dài và mục tiêu video.
```

### Tạo giọng đọc tiếng Việt

```text
Tôi muốn tạo giọng đọc tiếng Việt.
Hãy cho tôi biết repo có những cách nào: cách miễn phí/local và cách cần API key.
Hãy đề xuất cách dễ nhất cho người mới.
```

### Render video hoàn chỉnh

```text
Tôi muốn render video hoàn chỉnh.
Hãy kiểm tra giúp tôi cần chuẩn bị gì cho Remotion, Node.js, ffmpeg và các skill liên quan.
Nếu thiếu gì, hãy hướng dẫn từng bước thật đơn giản.
```

## 5. Khi agent yêu cầu cài đặt

Nếu agent nói cần cài thêm, bạn có thể hỏi lại:

```text
Trong các thứ đang thiếu, cái nào bắt buộc để dùng tính năng tôi muốn?
Cái nào có thể để sau?
Hãy chỉ hướng dẫn phần tối thiểu trước.
```

Nếu agent chuẩn bị chạy lệnh, hỏi:

```text
Lệnh này dùng để làm gì?
Có thay đổi file nào trong máy tôi không?
Nếu lỗi thì có khôi phục được không?
Giải thích ngắn gọn rồi chờ tôi đồng ý.
```

## 6. Khi đã sẵn sàng làm video

Prompt khởi động một job thực tế nên nói thẳng input bạn có và output bạn muốn:

```text
Tạo job mới.

Sử dụng script sau để đọc với giọng phù hợp của AusyncLab:

[Dán nội dung script vào đây]

Đây là những video thô mà bạn có thể sẽ cần phân tích và chọn cảnh phù hợp để cắt và đưa vào video đầu ra:

[Tải lên hoặc đính kèm video thô ở đây]

Yêu cầu video:
- Chọn các cảnh phù hợp nhất từ video thô.
- Cắt và sắp xếp cảnh theo nội dung script.
- Tạo giọng đọc bằng AusyncLab với voice phù hợp.
- Phụ đề đơn giản.
- Đọc đến từ nào thì highlight nền cho từ đó.
- Nếu thiếu API key, file đầu vào, hoặc công cụ render, hãy báo tôi biết trước khi làm tiếp.
```

Nếu app AI không hỗ trợ tải video trực tiếp vào chat, hãy copy video vào thư mục `raw_assets/`, rồi nói với agent:

```text
Tôi đã đặt video thô trong raw_assets. Hãy phân tích các video đó và chọn cảnh phù hợp cho job này.
```

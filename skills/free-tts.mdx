## Mục đích

Sinh giọng đọc tiếng Việt bằng VieNeu-TTS (local/free) để tạo `voice.wav` ổn định cho pipeline, đặc biệt khi không có `AUSYNCLAB_API_KEY` hoặc user yêu cầu không dùng AusyncLab.

## Khi nào invoke

- User yêu cầu free TTS/local TTS/VieNeu-TTS.
- `.env` không có `AUSYNCLAB_API_KEY` hoặc key rỗng.
- Cần tạo narration đầu vào cho `$word-timestamps-extractor` và render plan.

## Quy tắc chính

- Không yêu cầu hoặc in `AUSYNCLAB_API_KEY`.
- Giữ nguyên dấu tiếng Việt trong text narration.
- Nếu thiếu package `vieneu`, script có thể tự cài.

## Cách dùng nhanh

```text
$free-tts — tạo voice.wav miễn phí bằng VieNeu-TTS cho job hiện tại
```

Script trực tiếp:

```bash
.venv/bin/python skills/free_tts/scripts/free_tts.py \
  --text "Chào bạn. Đây là giọng đọc được tạo miễn phí bằng VieNeu-TTS." \
  --output-audio source/voice.wav \
  --output source/voice_selection.toml
```

Từ creative plan:

```bash
.venv/bin/python skills/free_tts/scripts/free_tts.py \
  --creative-plan jobs/<job_id>/source/creative_plan.toml \
  --output-audio jobs/<job_id>/source/voice.wav \
  --output jobs/<job_id>/source/voice_selection.toml
```

## Output

- Global: `source/voice.wav`, `source/voice_selection.toml`
- Theo job: `jobs/<job_id>/source/voice.wav`, `jobs/<job_id>/source/voice_selection.toml`

## Troubleshooting

- Thiếu `vieneu`: cài bằng `python -m pip install vieneu`.
- Text quá ngắn gây phát âm kém: ghép thành câu tự nhiên hơn.
- Tên riêng/English khó đọc: thử `--mode turbo`.

## Mục đích

Sinh narration `.wav` / `.mp3` bằng **VieNeu-TTS** khi không có `AUSYNCLAB_API_KEY`, hoặc khi user yêu cầu dùng free/local TTS thay AusyncLab.

Skill hỗ trợ:

- Preset voices của VieNeu-TTS (`Binh`, `Tuyen`, `Vinh`, `Doan`, `Ly`, `Ngoc`)
- Voice clone từ sample audio
- Registry `.shared` để đặt tên giọng dùng lại giữa các job
- Auto-install package `vieneu` nếu môi trường chưa có

## Khi nào được invoke

- User nói “tự động tạo giọng đọc cho...”
- `.env` không có `AUSYNCLAB_API_KEY`
- User yêu cầu “free TTS”, “VieNeu-TTS”, “local TTS”, hoặc “không dùng AusyncLab”
- Cần clone giọng từ sample `.wav` / `.mp3`

## Cách dùng nhanh

```text
$free-tts — tạo giọng đọc miễn phí cho script trong job hiện tại
```

Hoặc gọi script trực tiếp:

```bash
.venv/bin/python skills/free_tts/scripts/free_tts.py \
  --text-file jobs/<job_id>/source/narration.txt \
  --voice-id Binh \
  --output-audio jobs/<job_id>/source/voice.wav \
  --output jobs/<job_id>/source/voice_selection.toml
```

## Voice clone đặt tên

`.shared` nằm cùng cấp `.env` và dùng format:

```text
CON_VO=raw_assets/audio/voice_samples/CON_VO_sample.wav
```

Tên giọng phải là ASCII không dấu: `A-Z`, `a-z`, `0-9`, `_`, bắt đầu bằng chữ.

Lưu giọng:

```bash
.venv/bin/python skills/free_tts/scripts/free_tts.py \
  --save-voice-name CON_VO \
  --save-voice-audio raw_assets/audio/voice_samples/CON_VO_sample.wav
```

Dùng giọng đã lưu:

```bash
.venv/bin/python skills/free_tts/scripts/free_tts.py \
  --text-file jobs/<job_id>/source/narration.txt \
  --mode turbo \
  --voice-name CON_VO \
  --output-audio jobs/<job_id>/source/voice.wav
```

<Tip>
  `turbo` phù hợp khi clone giọng vì không cần `ref_text`; `standard` là mặc định cho preset voice.
</Tip>

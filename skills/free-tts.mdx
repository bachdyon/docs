## Mục đích

Sinh narration tiếng Việt (`.wav` / `.mp3`) bằng **VieNeu-TTS** khi không có `AUSYNCLAB_API_KEY`, hoặc khi user yêu cầu free/local TTS thay AusyncLab.

Skill hỗ trợ:

- Preset voices (`Binh`, `Tuyen`, `Vinh`, `Doan`, `Ly`, `Ngoc`)
- Voice clone từ sample audio
- Registry `.shared` để đặt tên giọng dùng lại giữa các job
- Tự cài package `vieneu` nếu môi trường chưa có

## Khi nào invoke

- User yêu cầu free TTS, local TTS, VieNeu-TTS hoặc không dùng AusyncLab.
- `.env` không có `AUSYNCLAB_API_KEY` hoặc key rỗng.
- Cần narration đầu vào cho `$word-timestamps-extractor` và render plan.
- User nói “tự động tạo giọng đọc cho…” hoặc cần clone giọng từ file `.wav` / `.mp3`.

## Quy tắc chính

- Không yêu cầu hoặc in `AUSYNCLAB_API_KEY`.
- Giữ nguyên dấu tiếng Việt trong text narration.

## Cách dùng nhanh

```text
$free-tts — tạo giọng đọc miễn phí bằng VieNeu-TTS cho job hiện tại
```

Script với `--text`:

```bash
.venv/bin/python skills/free_tts/scripts/free_tts.py \
  --text "Chào bạn. Đây là giọng đọc được tạo miễn phí bằng VieNeu-TTS." \
  --output-audio source/voice.wav \
  --output source/voice_selection.toml
```

Từ creative plan:

```bash
.venv/bin/python skills/free_tts/scripts/free_tts.py \
  --creative-plan jobs/<job_id>/source/creative_plan.toml \
  --output-audio jobs/<job_id>/source/voice.wav \
  --output jobs/<job_id>/source/voice_selection.toml
```

Preset voice + narration file:

```bash
.venv/bin/python skills/free_tts/scripts/free_tts.py \
  --text-file jobs/<job_id>/source/narration.txt \
  --voice-id Binh \
  --output-audio jobs/<job_id>/source/voice.wav \
  --output jobs/<job_id>/source/voice_selection.toml
```

## Voice clone và registry `.shared`

File `.shared` nằm cùng cấp `.env`, ví dụ:

```text
CON_VO=raw_assets/audio/voice_samples/CON_VO_sample.wav
```

Tên giọng: ASCII không dấu (`A-Z`, `a-z`, `0-9`, `_`), bắt đầu bằng chữ.

Lưu giọng:

```bash
.venv/bin/python skills/free_tts/scripts/free_tts.py \
  --save-voice-name CON_VO \
  --save-voice-audio raw_assets/audio/voice_samples/CON_VO_sample.wav
```

Dùng giọng đã lưu:

```bash
.venv/bin/python skills/free_tts/scripts/free_tts.py \
  --text-file jobs/<job_id>/source/narration.txt \
  --mode turbo \
  --voice-name CON_VO \
  --output-audio jobs/<job_id>/source/voice.wav
```

<Tip>
  `turbo` phù hợp khi clone giọng (không cần `ref_text`); `standard` là mặc định cho preset voice.
</Tip>

## Output

- Global: `source/voice.wav`, `source/voice_selection.toml`
- Theo job: `jobs/<job_id>/source/voice.wav`, `jobs/<job_id>/source/voice_selection.toml`

## Troubleshooting

- Thiếu `vieneu`: `python -m pip install vieneu`.
- Text quá ngắn gây phát âm kém: ghép thành câu tự nhiên hơn.
- Tên riêng / English khó đọc: thử `--mode turbo`.

## Cách invoke

Trong chat với Cursor/Codex/Claude Code, gõ ký hiệu `$<skill-name>` để gợi ý agent load skill đó. Ví dụ:

```text
$video-creative-planner — viết kịch bản TikTok 45s về buổi sáng yên tĩnh
```

Agent sẽ tự load `SKILL.md` tương ứng và làm theo. Không cần biết chi tiết bên trong.

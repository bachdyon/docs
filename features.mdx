## Tự động hoá pipeline video

VAS đóng gói **28 project skill** (trong `skills/<name>/SKILL.md`) + **1 skill bên thứ ba** (`$remotion-best-practices`). Mỗi skill là một module độc lập — agent đọc `SKILL.md` khi cần; bạn có thể bấm từng card bên dưới để xem tóm tắt trên docs hoặc mở [Tổng quan Skills](/skills/index).

## Vận hành đơn giản

<Columns cols={2}>
  <Card title="Auto installer" icon="rocket" href="/quickstart">
    Người mới có thể yêu cầu agent kiểm tra máy và chạy installer phù hợp, thay vì tự đọc từng bước cài đặt.
  </Card>
  <Card title="Watcher chạy 24/7" icon="rotate" href="/usage/status">
    Đăng ký với launchd (mac) / Task Scheduler (win) → tự bật cùng máy, tự restart khi crash. Đọc [`Status.command`](usage/status) để xem PID + lỗi gần nhất.
  </Card>
  <Card title="Idempotent" icon="recycle" href="/asset-index/idempotency">
    Cùng 1 file index 100 lần chỉ tốn quota AI **đúng 1 lần**. Watcher restart, copy file qua nhiều folder, scan lại toàn bộ — đều an toàn, không ngốn key.
  </Card>
  <Card title="Đa nền tảng" icon="globe" href="/asset-index/cross-platform">
    macOS (FSEvents), Windows (ReadDirectoryChangesW), Linux (inotify) — đều hoạt động. Có fallback `--polling` cho NFS / sandbox.
  </Card>
</Columns>

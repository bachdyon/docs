Repo dùng 4 API key, lưu trong file `.env` ở root. File mẫu là [`env.example`](https://github.com/bachdyon/video-automator-skills/blob/main/env.example).

<Warning>
  `.env` đã nằm trong [`.gitignore`](https://github.com/bachdyon/video-automator-skills/blob/main/.gitignore). **TUYỆT ĐỐI KHÔNG** commit file này — sẽ lộ key.
</Warning>

## Tạo file `.env`

```bash
cp env.example .env
```

Sau đó mở `.env` và điền giá trị từng biến.

## Bảng các biến

| Biến | Vai trò | Bắt buộc | Link tạo key |
| --- | --- | --- | --- |
| `OPENAI_API_KEY` | Embed text-embedding-3-small (asset index) + mọi skill dùng GPT | Có | [platform.openai.com/api-keys](https://platform.openai.com/api-keys) |
| `GEMINI_API_KEY` | Vision/multimodal analyzer cho asset index (image, video, audio) | Có | [aistudio.google.com/apikey](https://aistudio.google.com/apikey) |
| `AUSYNCLAB_API_KEY` | Sinh narration audio qua `$ausynclab-voice` | Tuỳ chọn | [ausynclab.io](https://ausynclab.io) |
| `AUSYNCLAB_VOICE_ID` | ID voice mặc định cho TTS | Tuỳ chọn | List voice trong AusyncLab dashboard |

<Note>
  **Auto installer** (`setup/Install.command` / `Install.bat`) sẽ prompt 2 key bắt buộc và tự ghi vào `.env`. AusyncLab key prompt sau, hoặc bỏ qua nếu không dùng voice TTS.
</Note>

## Ví dụ `.env` đầy đủ

```env
OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
GEMINI_API_KEY=AIzaSyxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
AUSYNCLAB_API_KEY=ausx-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
AUSYNCLAB_VOICE_ID=voice_001
```

## Verify key đã hoạt động

<AccordionGroup>
  <Accordion title="Test OPENAI_API_KEY">
    ```bash
    .venv/bin/python -c "import os; from openai import OpenAI; print(OpenAI(api_key=os.getenv('OPENAI_API_KEY')).models.list().data[0].id)"
    ```

    Output: tên model đầu tiên (ví dụ `gpt-4o`). Nếu lỗi 401 → key sai/hết quota.
  </Accordion>

  <Accordion title="Test GEMINI_API_KEY">
    ```bash
    .venv/bin/python -c "import os, google.generativeai as g; g.configure(api_key=os.getenv('GEMINI_API_KEY')); print([m.name for m in g.list_models()][:3])"
    ```

    Output: 3 model name đầu tiên. Nếu lỗi → key sai hoặc chưa enable Generative Language API trong AI Studio.
  </Accordion>
</AccordionGroup>

## Lưu ý bảo mật

- Mỗi key có quota riêng — kiểm tra dashboard nhà cung cấp trước khi index pool lớn.
- Quay key (rotate) định kỳ; nếu nghi lộ → revoke ngay từ dashboard và sinh key mới.
- Không paste key vào Issue, PR, hoặc chat public — dùng GitHub Secret nếu cần CI.
